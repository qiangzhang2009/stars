# 🔮 AI Fortune Telling Platform

An AI-powered divination and fortune-telling web application that combines Eastern metaphysics with Western mysticism.

## ✨ Features

- **Eastern Divination**: Bazi (Four Pillars), I Ching, Zhou Gong Dream Interpretation, Name Analysis, Fortune Drawing
- **Western Mysticism**: Tarot Cards, Horoscope, Numerology, Palmistry, MBTI Personality Test
- **AI-Powered**: Intelligent dialogue, personalized readings, real-time updates

## 🚀 Quick Start

```bash
# Install dependencies
cd frontend
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

## 🛠️ Tech Stack

- Vue 3
- Vite
- Element Plus
- Vue Router
- Vuex
- Vue I18n
- ECharts
